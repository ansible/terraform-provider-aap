# aap_eda_project (Resource)

Manages an EDA Project resource.

## Example Usage

### Basic EDA Project

```terraform
data "aap_organization" "default" {
  name = "Default"
}

resource "aap_eda_project" "example" {
  name            = "My EDA Project"
  description     = "An example EDA project"
  url             = var.project_repo_url
  organization_id = data.aap_organization.default.id
}
```

### EDA Project with SCM Branch

```terraform
resource "aap_eda_project" "with_branch" {
  name            = "EDA Project with Branch"
  description     = "EDA project using specific branch"
  url             = var.project_repo_url
  scm_branch      = "main"
  organization_id = 1
}
```

### EDA Project with Proxy

```terraform
resource "aap_eda_project" "with_proxy" {
  name            = "EDA Project with Proxy"
  description     = "EDA project using proxy server"
  url             = var.project_repo_url
  proxy           = "http://proxy.example.com:8080"
  organization_id = 1
}
```
