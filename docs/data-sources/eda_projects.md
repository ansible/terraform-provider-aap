# aap_eda_projects (Data Source)

Gets a list of EDA Projects with optional filtering.

## Example Usage

### Get All EDA Projects

```terraform
# Get all EDA projects
data "aap_eda_projects" "all" {}
```

### Filter by Organization

```terraform
data "aap_organization" "default" {
  name = "Default"
}

# Get all EDA projects in a specific organization
data "aap_eda_projects" "by_org" {
  organization_id = data.aap_organization.default.id
}
```

### Filter by Name

```terraform
# Get EDA projects with names containing "production"
data "aap_eda_projects" "production" {
  name_contains = "production"
}
```
